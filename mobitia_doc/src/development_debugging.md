# 4. 開発とデバッグ
本プロジェクトでは、システムの複雑な非同期挙動やリアルタイム性能を検証するために、ログ出力とトレーシングを活用しています。

## デバッグの基本方針
1. **コンソール出力**: リアルタイムな状態確認に使用します。
2. **システムトレーシング**: タイミング依存のバグやボトルネックの調査に使用します。
3. **シミュレーション**: 実機（LiDAR/モーター）なしでのロジック検証に使用します。
